# Weather Diary

Автор: [Чичкин Кирилл]

## Описание

"Weather Diary" - это приложение, которое позволяет пользователю записывать, просматривать и фильтровать записи о погоде. Программа позволяет добавлять информацию о погодных условиях и сохранять данные в формате JSON.

## Инструкция по запуску

1. Убедитесь, что Python установлен на вашей машине.
2. Установите необходимые библиотеки (если требуется):
   ```bash
   pip install tk
## Запустите приложение

python weather_diary.py

## Примеры использования

1. Введите дату в формате YYYY-MM-DD, температуру в °C, описание погоды и укажите, были ли осадки (да/нет).
2. Нажмите кнопку "Добавить запись" для добавления новой записи.
3. Используйте поля для фильтрации по дате и температуре, чтобы отобразить соответствующие записи.
